# lift-js
A JavaScript interpreter written in Python that supports primary features of ECMAScript 2015

## Set up virtualenv

```bash
virtualenv env
```

## Install requirements

```bash
source env/bin/activate
make install
```

## Run
```bash
make run
```

## Test
```bash
make test
```

## Lint
```bash
make lint
```

## Clean
```bash
make clean
```

## Generate docs
```bash
make doc
```

## Clean docs
```bash
make doc-clean
```

## Deactivate virtualenv
```bash
deactivate
```
